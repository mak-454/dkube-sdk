DKube API¶

This module defines APIs which can be used to interact with DKube platform. Using the SDK APIs all the different kind of dkube resources can be managed. Some network intensive functions are not available like downloading logs, exporting models to a local filesystem.

Import like below in your application. from dkube.sdk import *
